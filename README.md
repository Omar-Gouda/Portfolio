# Omar Gouda Portfolio

A personal portfolio website showcasing my skills, projects, and contact information as a Frontend Developer.

---

## About This Project

This portfolio highlights:

- **Skills Section:** Organized into Frontend, Frameworks, Tools, and Soft Skills with icons.
- **Projects Section:** Interactive project cards with hover/swipe effects to show multiple images.
- **Contact Section:** A functional contact form using EmailJS for sending messages.
- **Responsive Design:** Fully responsive for desktop, tablet, and mobile devices.
- **Clean UI/UX:** Focused on readability, modern design, and smooth interactions.

**Note:** I am still learning frontend development, so this project represents my current level of experience and practice.
